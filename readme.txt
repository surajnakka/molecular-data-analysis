The source code for each module can be found in this zip folder under 

"MSDataAnalysis/src/main/scala/" directory

Contents of Directory:

1.som.scala  - calculates sum of masses
2.soq.scala  - calculates sum of charges
3.moi.scala  - Moment of Inertia 
4.moiz.scala - Moment of Inertia over Z-Axis
5.dpm.scala  - Dipole Moment
6.com.scala  - Center of Mass


*A command line interactive interface can be started by clicking "MSDataAnalysis" file under "MSDataAnalysis" directory.But before,we need to set the path when running on a different machine.Present path is set with respect to the system we have run the application on.
